dotfiles
========

Mac branch.


vim
---

1. 系统依赖: ctags

2. mac安装ctags:

  - Download ctags from http://ctags.sourceforge.net/
  - extract it, and 

    ```
    ./configure
    sudo make install
    ```

3. 然后:

  ```
  $ make vim
  ```
